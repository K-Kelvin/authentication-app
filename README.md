# Authentication App

-   Frontend made with

    -   React
    -   Tailwindcss

-   Backend made with
    -   Nodejs
    -   Express

## How to get started locally

```bash
# To clone the repo
git clone https://github.com/K-Kelvin/authentication-app

# Install dependencies
npm install

### run the development server(windows)
npm start

### run the development server(linux)
npm run start-linux
```
